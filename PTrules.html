Periodic Table Law

- The arrangement rules of the elements:

  1. The chemistry elements are arranged from left to right and from top to bottom accordingly to their increasing dimension of atomic numbers.
  2. The horizontal rows (periods) have the same number of electron subshells, namely the fourth period that includes from potassium to kryton.
  3. The vertical columns (groups) have the same number of valency electrons, for example Lithium is similar to sodium, beryllium is similar to magnesium, and so on.
  4. Element blocks:
    • ordinal number = atomic number
    • the same period => the same number of electron subshells
    • the ordinal number of group = the number of valency electrons

- Some usual laws of PT

Note: these laws may be wrong with some elements or in some cases

  • In a period:
    • the same number of electron subshells
    • increasing atomic charge
    • => gravitational force of the atomic increases
  • In a group:
    • The atomics charge increases => increasing screening constant (denoted by s, S or Ïƒ), which relates the effective and actual nuclear charges as

      • Z(eff) or Z* = Z - s

    • While the number electron subshells increase dramatically at the same time
    • => atomic radius increases due to Slater's formula for computing the atomic orbital radius:

      • r(AO) = (n*)2/(Z*)2

    • atomic charge
    • screening constant of inner electrons
    • pushing interaction from paired electrons
